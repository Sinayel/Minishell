## Run Locally

Clone the project

```bash
  git clone https://github.com/Sinayel/Minishell.git
```

Go to the project directory

```bash
  cd Minishell
```
Make le Minishell
```bash
  Make
```
Lancer le Minishell
```bash
  ./Minishell [instruction]
```
